# ISTQB Study Game

A fullstack study app for the ISTQB Certified Tester Foundation Level (CTFL v4.0) syllabus. Three game modes — memory match, spaced-repetition flashcards, and drag-to-categorize — with progress persistence, an installable PWA, and a multi-layered test suite.

## Status

🚧 Under construction. See the project plan for the roadmap.

## Stack

- **Frontend:** Vue 3 + Vite + TypeScript, Tailwind CSS, Pinia, PWA via `vite-plugin-pwa`
- **Backend:** FastAPI (Python 3.11+), SQLAlchemy 2.0, Alembic, JWT auth
- **Database:** PostgreSQL (production) / SQLite (development)
- **Testing:** Pytest, Vitest, Playwright, Axe (accessibility)
